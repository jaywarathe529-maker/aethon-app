# Aethon — Setup Guide

## Files in this package

```
aethon-app/
├── src/
│   ├── App.jsx              ← Routes between auth and main app
│   ├── Aethon.jsx           ← Main app (all 6 features)
│   ├── main.jsx             ← Entry point
│   ├── index.css            ← Global styles
│   ├── pages/
│   │   └── Auth.jsx         ← Sign In / Sign Up / Forgot Password
│   ├── context/
│   │   └── AuthContext.jsx  ← Login state management
│   └── firebase/
│       └── config.js        ← Firebase connection
├── public/
│   └── icons/               ← App icons (generated by script)
├── index.html               ← HTML with PWA meta tags
├── vite.config.js           ← Vite + PWA plugin config
├── package.json             ← Dependencies
├── generate_icons.py        ← Generates all app icons
└── .env.local               ← Your Firebase keys (fill this in)
```

---

## Step 1 — Install dependencies

```bash
npm install
```

---

## Step 2 — Set up Firebase (free)

1. Go to https://console.firebase.google.com
2. Click "Add project" → name it "aethon-app"
3. Disable Google Analytics (not needed) → Create project
4. Click the Web icon ( </> ) to add a web app
5. Name it "aethon" → Register app
6. Copy the firebaseConfig values shown

Now open .env.local and fill in your values:

```
VITE_FIREBASE_API_KEY=AIzaSy...
VITE_FIREBASE_AUTH_DOMAIN=aethon-app.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=aethon-app
VITE_FIREBASE_STORAGE_BUCKET=aethon-app.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=123456789
VITE_FIREBASE_APP_ID=1:123456:web:abc123
```

---

## Step 3 — Enable Firebase Authentication

1. In Firebase Console → Authentication → Get started
2. Enable "Email/Password"
3. Enable "Google"
4. For Google: set your support email

---

## Step 4 — Enable Firestore Database

1. Firebase Console → Firestore Database → Create database
2. Choose "Start in test mode"
3. Pick a region closest to India (asia-south1)
4. Click Enable

---

## Step 5 — Generate app icons

```bash
python3 generate_icons.py
```

You should see 8 icon files created in public/icons/

---

## Step 6 — Run locally to test

```bash
npm run dev
```

Open http://localhost:5173
You should see the Sign In screen.
Create an account and test the full flow.

---

## Step 7 — Build for production

```bash
npm run build
```

---

## Step 8 — Deploy to Vercel

1. Push all files to GitHub:
```bash
git add .
git commit -m "Add PWA + Firebase auth"
git push
```

2. Go to vercel.com → your project → Settings → Environment Variables
3. Add ALL the VITE_FIREBASE_* variables from your .env.local
4. Redeploy

---

## Step 9 — Add Firebase authorized domain

1. Firebase Console → Authentication → Settings → Authorized domains
2. Add your Vercel URL (e.g. aethon-app.vercel.app)

---

## Testing PWA on Android

1. Open your Vercel URL in Chrome on Android
2. Tap the 3-dot menu
3. Tap "Add to Home screen"
4. App installs with your icon

## Testing PWA on iPhone

1. Open your Vercel URL in Safari on iPhone
2. Tap the Share button (box with arrow)
3. Tap "Add to Home Screen"
4. App installs with your icon
